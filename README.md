# Convertor X v 1.0

Convertor X es una aplicación web ligera, moderna y potente diseñada para la conversión de imágenes directamente en el navegador. No requiere instalación de software ni conexión a un servidor para procesar los archivos, garantizando velocidad y privacidad.

# 🚀 Características Principales


    * Conversión Multi-formato: Soporte para convertir imágenes a los formatos más utilizados: PNG, JPG, WEBP e ICO.
    * Procesamiento por Lotes: Capacidad para seleccionar y convertir múltiples imágenes simultáneamente, ahorrando tiempo al usuario.
    * Control de Calidad (JPG): Slider interactivo para ajustar la calidad de compresión en imágenes JPG (de 1 a 100).
    * Generador de Iconos (ICO): Opciones avanzadas para crear iconos, permitiendo seleccionar múltiples resoluciones (16x16, 32x32, 48x48, 64x64, 128x128, 256x256).
    * Interfaz Moderna (UI/UX):
    * Diseño limpio con tarjetas y sombras suaves.
    * Splash Screen: Animación de carga inicial profesional.
    * Notificaciones de éxito animadas.
    * Barra de estado informativa.
    * Diseño Responsivo: Interfaz adaptable a dispositivos móviles, tablets y escritorio.
    * Privacidad Total: Todo el procesamiento se realiza localmente en el navegador del usuario; ningún archivo se sube a servidores externos.

# 🛠️ Tecnologías Utilizadas

Este proyecto está construido con tecnologías web estándar, sin dependencias de frameworks pesados, lo que lo hace extremadamente ligero.

    * HTML5: Estructura semántica y uso de la API de archivos (File API) para manejo local de documentos.
    * CSS3: Estilos avanzados mediante Flexbox y Grid.
    * Animaciones con @keyframes.
    * Diseño responsivo con Media Queries.
    * Variables CSS para una paleta de colores consistiente.
    * JavaScript (Vanilla JS): Lógica de aplicación pura sin librerías externas.
    * Uso de Canvas API para el procesamiento y redimensionamiento de imágenes.
    * Manipulación del DOM para una experiencia de usuario dinámica.

# 💻 Instalación y Uso

Al ser una aplicación web autocontenida en un solo archivo, su instalación es muy sencilla.
Opción A: Uso Local

    Clona el repositorio o descarga el archivo index.html.

    git clone https://github.com/Pr0fEs0r-X/convertor-x.git

    Haz doble clic en el archivo index.html o ábrelo con tu navegador web preferido (Chrome, Firefox, Edge, Safari).
    ¡Listo! La aplicación se cargará y podrás empezar a convertir imágenes.

#### Opción B: Despliegue en la Web

Puedes alojar este archivo en cualquier servidor web estático o servicio de hosting gratuito como:

    GitHub Pages
    Netlify
    Vercel
